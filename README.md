# ระบบคำนวณค่าจัดส่งสินค้า (Shipping Cost Calculator)

## ภาพรวมของโปรแกรม

โปรแกรม `ShippingCalculator.java` เป็นระบบคำนวณค่าจัดส่งสินค้าที่พัฒนาด้วยภาษา Java โดยคำนวณค่าจัดส่งตามเงื่อนไขต่างๆ ได้แก่ ปลายทาง น้ำหนักสินค้า บริการด่วน และส่วนลดสมาชิก VIP

---

## โครงสร้างโปรแกรม

### 1. ค่าคงที่ (Constants) - บรรทัดที่ 12-26

```java
// ค่าจัดส่งสำหรับกรุงเทพฯ และปริมณฑล (บาท)
private static final int BKK_RATE_0_1_KG = 40;      // 0-1 กก. = 40 บาท
private static final int BKK_RATE_1_3_KG = 60;      // 1-3 กก. = 60 บาท
private static final int BKK_RATE_3_5_KG = 80;      // 3-5 กก. = 80 บาท
private static final int BKK_RATE_OVER_5_KG = 100;  // มากกว่า 5 กก. = 100 บาท

// ค่าจัดส่งสำหรับต่างจังหวัด (บาท)
private static final int PROVINCIAL_RATE_0_1_KG = 60;       // 0-1 กก. = 60 บาท
private static final int PROVINCIAL_RATE_1_3_KG = 90;       // 1-3 กก. = 90 บาท
private static final int PROVINCIAL_RATE_3_5_KG = 120;      // 3-5 กก. = 120 บาท
private static final int PROVINCIAL_RATE_OVER_5_KG = 150;   // มากกว่า 5 กก. = 150 บาท

// ค่าบริการเพิ่มเติม
private static final int EXPRESS_FEE = 30;              // ค่าบริการด่วน 30 บาท
private static final double VIP_DISCOUNT_RATE = 0.20;   // ส่วนลด VIP 20%
```

**อธิบาย:** กำหนดค่าคงที่สำหรับอัตราค่าจัดส่งตามน้ำหนักและปลายทาง รวมถึงค่าบริการด่วนและอัตราส่วนลด VIP

---

### 2. เมธอด `calculateBaseShipping()` - บรรทัดที่ 34-62

**หน้าที่:** คำนวณค่าจัดส่งพื้นฐานตามน้ำหนักและปลายทาง

**ขั้นตอนการทำงาน:**

1. **ตรวจสอบน้ำหนัก** - ถ้าน้ำหนัก ≤ 0 จะแจ้งข้อผิดพลาด
2. **ตรวจสอบปลายทาง** - แยกเงื่อนไขตามปลายทาง
3. **คำนวณตามช่วงน้ำหนัก:**

| ช่วงน้ำหนัก | กรุงเทพฯ/ปริมณฑล | ต่างจังหวัด |
|------------|-----------------|------------|
| 0-1 กก. | 40 บาท | 60 บาท |
| 1-3 กก. | 60 บาท | 90 บาท |
| 3-5 กก. | 80 บาท | 120 บาท |
| > 5 กก. | 100 บาท | 150 บาท |

**ผังการทำงาน:**
```
เริ่มต้น
    ↓
น้ำหนัก > 0 ? → ไม่ → แจ้งข้อผิดพลาด
    ↓ ใช่
ปลายทางกรุงเทพฯ?
    ↓ ใช่              ↓ ไม่ (ต่างจังหวัด)
คำนวณอัตรา BKK    คำนวณอัตรา Provincial
    ↓                  ↓
      คืนค่าค่าจัดส่ง
```

---

### 3. เมธอด `calculateTotalShipping()` - บรรทัดที่ 72-88

**หน้าที่:** คำนวณค่าจัดส่งรวมทั้งหมด (รวมบริการด่วนและส่วนลด VIP)

**ขั้นตอนการทำงาน:**

1. **คำนวณค่าจัดส่งพื้นฐาน** - เรียกใช้ `calculateBaseShipping()`
2. **เพิ่มค่าบริการด่วน** - ถ้าเลือกบริการด่วน +30 บาท
3. **คำนวณส่วนลด VIP** - ถ้าเป็นสมาชิก VIP ลด 20%

**สูตรการคำนวณ:**
```
ค่าจัดส่งรวม = (ค่าจัดส่งพื้นฐาน + ค่าบริการด่วน) × (1 - อัตราส่วนลด VIP)
```

**ตัวอย่าง:**
- สินค้า 2 กก. ส่งกรุงเทพฯ + ด่วน + VIP
- ค่าจัดส่งพื้นฐาน = 60 บาท
- ค่าบริการด่วน = 30 บาท
- รวมก่อนลด = 90 บาท
- ส่วนลด VIP 20% = 18 บาท
- **ค่าจัดส่งสุทธิ = 72 บาท**

---

### 4. เมธอด `printShippingDetails()` - บรรทัดที่ 93-123

**หน้าที่:** แสดงรายละเอียดการคำนวณค่าจัดส่งอย่างละเอียด

**ขั้นตอนการทำงาน:**

1. **แสดงหัวข้อ** - พิมพ์กรอบหัวข้อ
2. **แสดงข้อมูลที่รับเข้า:**
   - น้ำหนักสินค้า
   - ปลายทาง
   - สถานะบริการด่วน
   - สถานะสมาชิก VIP
3. **แสดงการคำนวณ:**
   - ค่าจัดส่งพื้นฐาน
   - ค่าบริการด่วน (ถ้ามี)
   - ส่วนลด VIP (ถ้ามี)
4. **แสดงค่าจัดส่งรวม**

**ตัวอย่างผลลัพธ์:**
```
========================================
       รายละเอียดค่าจัดส่งสินค้า
========================================
น้ำหนักสินค้า: 2.00 กิโลกรัม
ปลายทาง: กรุงเทพฯ และปริมณฑล
บริการด่วน: ใช่
สมาชิก VIP: ใช่
----------------------------------------
ค่าจัดส่งพื้นฐาน: 60 บาท
ค่าบริการด่วน: +30 บาท
ส่วนลด VIP (20%): -18.00 บาท
----------------------------------------
ค่าจัดส่งรวม: 72.00 บาท
========================================
```

## สรุปการไหลของข้อมูล (Data Flow)

```
ข้อมูลนำเข้า          การประมวลผล              ผลลัพธ์
─────────────────────────────────────────────────────────────
น้ำหนัก (กก.)    ──┐
                   │
ปลายทาง         ──┼──→ calculateBaseShipping() ──→ ค่าจัดส่งพื้นฐาน
                   │              │
บริการด่วน      ──┼──────────────┼──→ + EXPRESS_FEE (ถ้าเลือก)
                   │              │
สมาชิก VIP      ──┴──────────────┴──→ × (1-0.20) (ถ้าเป็น VIP)
                                              │
                                              ↓
                                      ค่าจัดส่งรวมสุทธิ
```
